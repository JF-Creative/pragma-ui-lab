# PRAGMA UI LAB

A collection of reusable frontend modules built as part of the PRAGMA ecosystem.

## Purpose

This repository works as a small laboratory for experimenting with visual and interactive UI components that can later be reused in different digital environments.

Examples of use cases include:

- websites
- interactive widgets
- dashboards
- digital media interfaces

## Repository structure

modules/

Each module lives in its own folder and contains its own HTML, CSS and JavaScript files.

Example modules that may appear in this repository:

- animated login
- minimal slider
- event countdown
- information widgets

## Status

Early stage. Modules will be added progressively as experiments and reusable components.
